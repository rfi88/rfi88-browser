<h3>📦 Install on Fedora 40/41/42/43</h3>
<pre><code>sudo dnf install https://github.com/rfi88/rfi88-browser/releases/download/v2.2/rfi88-browser-2.2-1.fc43.noarch.rpm</code></pre>

<h3>🚀 Update</h3>

<h3>🗑️ Uninstall</h3>
